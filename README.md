## 安裝

Node.js 版本建議為：v20.10.0 以上

### 取得專案

```
git clone https://github.com/tim8076/TaiwanBusApp.git
```

### 移動到專案內

```
cd TaiwanBusApp
```

### 安裝套件

```
npm install
```

### 運行專案

```
npm run dev
```

### 開啟專案

```
http://localhost:5173/#/
```
